Position

value : absolute, fixed, relative, sitcky, static

Absolute(절대좌표)

-콘텐츠 위치 좌표(left,right,top,bottom) -스크롤에 반응함 -다른 콘텐츠화 겹칠 수 있음 -사용예) 공지사항,이벤트,배너와 같은 곳에 사용

Relative(상대좌표)

-콘텐츠 위치 좌표(left,right,top,bottom) -스크롤에 반응함 -위치가 달라지더라도 콘텐츠 영역이 남아 있음 -콘텐츠를 겹치기 위해 사용하지 않음 -좌표값은 콘텐츠가 위치하고 있는 영역으로 설정됨(브라우저 X)

Relative-Absolute(상대좌표)

-부모(relative), 자손(absolute) -부모의 좌표 영역에서 자손이 위치값을 가짐 -사용 예) 이미지 위에 설명글....
absolute

Fixed(고정좌표)

-absolute와 동일하나 스크롤에 영향을 받지 않음 -항상 고정될 수 있는 콘텐츠(헤더, 사이드 메뉴...)

sticky(상대&고정좌표)

사이드 메뉴 또는 콘텐츠

sticky는 부모의 영역 안에서 고정값을 갖음 그외 relative와 동일하게 작동됨
